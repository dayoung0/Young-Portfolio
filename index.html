Ambition

This is one of my most ambitious projects to date. This is an incremental game based around fighting off a Zombie Plague!



Using HTML CSS and JS I created a multi-level clicker game. Using your mouse and clicking on the weapon you are tasked with clicking the mouse until all the Zombies are gone.
